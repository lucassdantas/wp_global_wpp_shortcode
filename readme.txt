== Description ==
Este plugin permite que você adicione um link direto para o WhatsApp em seu site WordPress usando shortcodes, facilitando a comunicação com seus visitantes. 

Você pode usar o shortcode [ld_whatsapp_link] para gerar um link para enviar uma mensagem via WhatsApp, e o [ld_show_whatsapp_number] para exibir o número configurado diretamente no seu conteúdo. 

Além disso, é possível personalizar a mensagem do link através do parâmetro 'text' no shortcode.

== Installation ==
1. Faça o upload da pasta `global_wpp_shortcode` para o diretório de plugins no seu servidor WordPress.
2. Ative o plugin através do menu 'Plugins' no WordPress.
3. Acesse a página de configurações do plugin e insira o número do WhatsApp.
4. Utilize os shortcodes `[ld_whatsapp_link]` e `[ld_show_whatsapp_number]` no seu site.

== Usage ==
Para utilizar o plugin, insira o seguinte shortcode em qualquer lugar do seu site:

- Para criar um link para WhatsApp:
  [ld_whatsapp_link text='Sua mensagem personalizada']
- Para exibir o número de WhatsApp configurado:
  [ld_show_whatsapp_number]

== Frequently Asked Questions ==

= Como posso alterar o número do WhatsApp? =
Acesse a página de configurações do plugin no painel do WordPress, onde você pode configurar o número de telefone.

= Posso usar uma mensagem personalizada no link do WhatsApp? =
Sim, você pode personalizar a mensagem passando o parâmetro 'text' no shortcode:  
[ld_whatsapp_link text="Sua mensagem aqui"]
